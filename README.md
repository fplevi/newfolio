# newfolio

Personal portfolio built entirely without Figma — no design tool, no visual editor.

## About the project

This portfolio was conceived and built exclusively through the terminal and AI collaboration. Every decision — layout, typography, spacing, interactions, copy — was made through conversation and code, with no mockups or wireframes created beforehand.

The images used across the case studies were also generated with AI tools, not exported from any design software.

## Stack

- Vanilla HTML, CSS, and JavaScript
- No frameworks, no build tools, no dependencies
- Contact form via [Formspree](https://formspree.io)
- Hosted on GitHub Pages

## How it was built

- **Code**: written and iterated entirely via terminal using [Claude Code](https://claude.ai/code) (Anthropic CLI)
- **Images**: generated with AI image tools, then optimized and exported as PNG
- **No Figma**: zero design files — layout and visual decisions made directly in code
